## The 45-Minute Framework

```
0-5 min:   Clarify requirements
5-10 min:  Estimate scale
10-20 min: High-level design
20-35 min: Deep dive (database, algorithms, bottlenecks)
35-40 min: Discuss tradeoffs
40-45 min: Handle follow-up questions
```

---

## Success Criteria

You've mastered Day 29 when you can:
- [ ] Complete a system design in 45 minutes
- [ ] Estimate scale correctly (within 2x)
- [ ] Identify the key bottleneck in your design
- [ ] Discuss tradeoffs without prompting
- [ ] Handle "what if scale is 10x?" questions

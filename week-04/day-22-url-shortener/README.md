# Day 22: URL Shortener — Complete Design

## "A Classic Interview Question Done Right"

**Estimated Time**: 2 hours  
**Difficulty**: Intermediate  
**Prerequisites**: Weeks 1–3 complete

---

## Learning Objectives
- Apply the complete system design framework to a real problem
- Design a production-ready URL shortener

- Scale from 1,000 to 1,000,000,000 URLs

## Quick Summary

The URL shortener is the "Hello World" of system design interviews. It's simple enough to design in 45 minutes but rich enough to reveal deep understanding. Today you'll design it completely — not just the happy path, but analytics, custom aliases, expiration, and scaling.
